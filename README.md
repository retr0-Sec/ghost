# 👻 Ghost - Voice Assistant Project

Projeto de assistente de voz desenvolvido em Python, com foco em automação, reconhecimento de voz e síntese de fala.

---

## 🚀 Sobre o projeto

O **Ghost** é um assistente de voz local que utiliza reconhecimento de fala e resposta por áudio para executar comandos e automações no sistema.

O objetivo é criar uma base extensível para:
- automação de tarefas
- controle por voz
- execução de comandos no sistema
- experimentos com IA local

---

## ⚙️ Tecnologias utilizadas

- Python 3.x
- SpeechRecognition
- Numpyshell
- Piper TTS
- Subprocess
- Poetry (gerenciamento de dependências)

---

## 📦 Instalação

### 1. Clone o repositório
```bash
git clone https://github.com/retr0-Sec/ghost.git
cd ghost
```
### 2 Instale o poetry (Caso não tenha)
```bash
pip install poetry
poetry install       #Instale as depedências
poetry run python main.py         # Rode o programa
```
## 3 Uso direto com python
```bash
python3 main.py
```
